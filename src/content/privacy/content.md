Cette politique de confidentialité explique comment je gére les informations que vous fournissez via mon formulaire de contact.

## Informations Collectées

Lorsque vous utilisez mon formulaire de contact, je collecte :
- Votre nom et prénom
- Votre adresse e-mail
- Le contenu de votre message

## Utilisation de Vos Informations

J'utilise ces informations uniquement pour :
- Répondre à vos demandes
- Maintenir un historique des communications

## Stockage et Sécurité des Données

- Vos soumissions via le formulaire de contact ne sont pas stockées dans une base de données
- Je ne partage pas vos informations avec des tiers

## Vos Droits

Vous avez le droit de :
- Demander l'accès à vos données
- Demander la suppression de vos données

## Me Contacter

Si vous avez des questions sur ma politique de confidentialité, veuillez me contacter via le site web.
